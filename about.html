SmartRoad Road Hazard Reporting System

Academic Project

SmartRoad is a crowdsourcing road hazard reporting system that allows users to submit road-related complaints through a mobile application. Administrators can review reports, monitor hazard locations, update report status, manage registered users and maintain report records through a web-based admin panel.

Project Objective

To provide a centralized platform for reporting and managing road hazards, allowing users to submit reports with images and location details while enabling administrators to monitor, update and organize reports efficiently.
